# Лабораторная работа №1: Основы Qt

Данная работа состоит из двух заданий, демонстрирующих возможности библиотеки Qt для создания графических интерфейсов (GUI) с использованием C++.

## Задание 1: Прокрастинация

Приложение для эффективного повторения учебных материалов (билетов) перед экзаменом. Программа позволяет отслеживать прогресс изучения каждого билета и выбирать следующий вопрос случайным образом.

### Функциональные возможности:
- **Управление количеством**: `QSpinBox` позволяет задать общее число билетов (от 1 до 1000). При изменении количества список сбрасывается к дефолтным значениям.
- **Список прогресса**: `QListWidget` отображает все билеты с цветовой индикацией статуса:
    - **Серый**: билет еще не повторялся.
    - **Желтый**: билет требует повторного прохода.
    - **Зеленый**: билет полностью изучен.
- **Детальный просмотр**: При выборе билета отображается его номер, текущее имя и статус.
- **Редактирование**: Поле `QLineEdit` позволяет переименовать билет (при нажатии Enter), а `QComboBox` — вручную изменить его статус.
- **Навигация**:
    - Кнопка **"Предыдущий"** позволяет вернуться к последнему просмотренному билету.
    - Кнопка **"Следующий случайный"** выбирает новый билет из тех, что еще не выучены (статус "Дефолтный" или "Желтый").
- **Статистика**: Два прогресс-бара отображают общий охват билетов и процент полностью выученного материала.
- **Интерактивы**: Двойной клик по билету в списке быстро переключает его статус (Дефолт -> Зеленый -> Желтый).

## Задание 2: Калькулятор Отчисления ("Свободное плавание")

Творческое задание: интерактивный калькулятор, оценивающий риск отчисления студента на основе различных факторов.

### Задействованные виджеты:
1. **QSlider**: выбор количества академических задолженностей (долгов).
2. **QCheckBox**: наличие "автоматов" по предметам.
3. **QRadioButton**: выбор формы обучения (Бюджет / Платное).
4. **QComboBox**: выбор среднего количества часов сна в сутки.
5. **QLineEdit**: ввод фамилии студента (для персонализированного вердикта).
6. **QProgressBar**: визуализация уровня риска (меняет цвет на красный при критических значениях).

### Логика работы:
Программа динамически вычисляет вероятность отчисления. Например, наличие долгов и недостаток сна повышают риск, в то время как "автоматы" и платная форма обучения (традиционно) его снижают. Результат выводится в виде текстового вердикта и заполнения шкалы риска.

## Особенности реализации (Design & UI)

Приложение выполнено в современном "Premium" стиле с использованием **Qt Style Sheets (QSS)**:
- **Цветовая палитра**: Светлая профессиональная тема с акцентами на корпоративный синий (#007bff).
- **Читаемость**: Шрифт билетов всегда черный для максимальной контрастности на любом фоне (серый/желтый/зеленый).
- **Адаптивность**: Все элементы управления размещены с использованием `QVBoxLayout`, `QHBoxLayout` и `QFormLayout`, что обеспечивает корректное отображение при любом размере окна.
- **Визуальные эффекты**: Скругленные углы (border-radius), мягкие тени и увеличенные шрифты для важных заголовков.
